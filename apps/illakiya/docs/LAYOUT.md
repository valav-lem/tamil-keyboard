# Tholkappiam Layout (PM0100) Draft

## Core Principle: Paired Units (Kuril-Nedil)

The keyboard is organized by sound pairs.

### Vowels (Uyir)
- **Unit 1:** A (அ) / Aa (ஆ)
- **Unit 2:** I (இ) / Ii (ஈ)
- **Unit 3:** U (உ) / Uu (ஊ)
- **Unit 4:** E (எ) / Ee (ஏ)
- **Unit 5:** Ai (ஐ)
- **Unit 6:** O (ஒ) / Oo (ஓ)
- **Unit 7:** Au (ஔ)

### Consonants (Mei) - Proposed Grouping
*Should we group by Vallinam (Hard), Mellinam (Soft), Idaiyinam (Medium)?*

**Vallinam:**
- K (க்) / C (ச்) / T (ட்) / Th (த்) / P (ப்) / R (ற்)

**Mellinam:**
- Ng (ங்) / Nj (ஞ்) / N (ண்) / N (ந்) / M (ம்) / N (ன்)

**Idaiyinam:**
- Y (ய்) / R (ர்) / L (ல்) / V (வ்) / Zh (ழ்) / L (ள்)
